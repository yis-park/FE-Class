블록 요소 / 자동으로 숫자가 부여 됨.

  1. ol과 li 사이엔 다른 태그가 올 수 없음.
  2. li태그 안에는 다양한 태그 작성 가능.
  3. ol과 li는 함께 사용. 둘 중 하나만 사용할 수 없음.
  4. 꼭 순서가 필요한 리스트에 작성.
  5. start 속성 작성 시 작성한 숫자부터 시작.
  1. 감자를 1/4 크기로 자른다.
  2. 부드러워질 때까지 15-20분 동안 소금물에 서서히 끓인다.
  3. 감자의 물을 따라내고 으깬다.
  4. 우유,버터,육두구를 데운다.
  5. 우유를 혼합물에 넣고 섞는다.
